Project: Typing speed

Description

The aim of this project is to create a webpage where the user can check his typing speed. The user is given a text and they must type it fully with as less mistakes as possible.

Various statistics can be computed:

  • CPM (Characters per minute): number of alphanumeric characters per minute
  • WPM (Words per minute): number of words per minute. A word can be defined as having, on average, 5 characters.
  • Accuracy: ratio between the number of valid characters vs the total number of typed characters.

You can see an example of the expected result in the video below. The goal is not to reproduce this example, but to design your own webpage and implement your own ideas. Any personal initiative is strongly encouraged.

Milestones

Here are the main steps to help you create a basic typing speed test. This is to get you started, you are encouraged to go further than those steps.

Steps:

  • Setup the HTML and CSS file. Create a basic webpage with the HTML tags required for typing and displaying information.
  • Handling user input. Detect when the user types, and compute statistics from it. There is no need to check what the user types against the text at this stage.
  • Check keyboard input versus text. Add a text in the HTML page and check that the user input is valid. Update statistics accordingly.
